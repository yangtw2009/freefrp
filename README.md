# 第一步獲取安裝軟件

``` wget http://yang.ddns.net:8000/yangfrp.sh ```

接者它應該會請你輸入服務器信息
以下為服務器信息

位置:yang.ddns.net
端口:7000
內網端口:請輸入你需要轉發的本地端口
外網端口:請輸入你要使用的外網端口可以到-->[這裡](http://yang.ddns.net:9010)
